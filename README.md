# 📌 Smart Wallpaper Manager

Smart Wallpaper Manager is a Python desktop application that allows users to search, download, and set wallpapers automatically.

The application uses the Unsplash API to fetch high-quality wallpapers based on a user-provided topic.

## 🚀Features
- Search wallpapers by topic
- Download high-resolution wallpapers
- Automatically set wallpaper on Windows
- Save download history
- Simple GUI built with Tkinter

## 🛠 Tech Stack
- Python
- Tkinter
- Requests
- Unsplash API
- File Handling

## ▶ How to Run

1. Clone the repository
2. Install dependencies

```bash
pip install requests
```

3. Run the program
```bash
python main.py
```
